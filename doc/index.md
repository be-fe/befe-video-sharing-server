# 简易团队视频分享平台

## 为什么要做这套系统

- 视频Demo能极好的支持团队中传递知识与经验
- 现行没有合适的支持视频分享的机制与系统
- 现代浏览器对html video的支持, 使快速分享视频的条件已经具备
